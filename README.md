# Travel Booking Platform

A full-stack travel booking application built using Ansible, Kubernetes, and CI/CD pipelines.

## Features
- Flight, hotel, and car booking
- Admin dashboard for management
- Frontend: React (Vite)
- Backend: Spring Boot
- Deployment: Kubernetes + Ansible automation
